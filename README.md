# Playwright SDET Portfolio

This repository contains end-to-end UI automation tests built using Playwright (JavaScript).

## What is tested
- Login (valid and invalid)
- Locked out user validation
- Logout
- Add to cart

## Tech Stack
- Playwright
- JavaScript
- Node.js

## How to run tests

1. Install dependencies:
npm install

2. Run all tests:
npx playwright test

3. Run tests in browser:
npx playwright test --headed

4. View test report:
npx playwright show-report
